# ShopSmart – E-commerce Interface (Lab 5 Capstone)

## Project Theme
E-commerce storefront for everyday essentials (electronics, home, and accessories).

## Sections Implemented
- Semantic `header` with logo, brand, navigation, and “Skip to main content” link.
- Hero/banner with title, tagline, CTA buttons, and animated promotional image.
- Products area:
  - Sidebar `aside` with visual filters (category, price, sort).
  - Product list/grid using CSS Grid with responsive cards.
- Deals section highlighting bundle offers.
- Testimonials section with customer feedback cards.
- Contact section with details and a fully labeled inquiry form.
- Footer with about text, navigation links, and SVG social icons.

## Layout & Responsiveness
- Flexbox used for header, hero actions, footer, and other alignments.
- CSS Grid used for:
  - Product cards
  - Deals
  - Testimonials
  - Main content layouts
- Breakpoints:
  - Desktop: ≥ 1024px (multi-column grid, sticky filters).
  - Tablet: ≤ 768px (stacked layout, 2-column grids).
  - Mobile/small phones: ≤ 480px (single-column grids, stacked header).
